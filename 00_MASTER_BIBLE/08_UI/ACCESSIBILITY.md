ACCESSIBILITY
Purpose
Accessibility defines the standards and features that ensure ROOTS is playable and enjoyable by the widest possible audience, including players with disabilities, different abilities, and diverse needs.

Accessibility Philosophy
Accessibility is not a feature bolted on at the end — it is integral to good design.
Every player deserves to experience the story and world of ROOTS, regardless of ability.

Vision & Colorblindness
Text & UI Contrast

Minimum contrast ratio: 4.5:1 for normal text, 3:1 for large text (WCAG AA standard).
No critical information conveyed by color alone.
Text always readable against background (no light text on light background, etc.).


Colorblind Modes
Supported colorblind vision types:

Deuteranopia (red-green, common)
Protanopia (red-green, less common)
Tritanopia (blue-yellow, rare)

Adjustments:

UI colors shifted to accommodate color vision deficiency.
Icons include patterns or symbols (not just color) to distinguish meaning.
Settings option to enable colorblind mode (applies globally).


Text Size & Font

Default font size: 16–18 pt minimum for readable text.
UI Scale option: 80% / 100% / 120% (adjusts all UI text size).
No thin or decorative fonts (sans-serif preferred for readability).
High contrast between text and background always maintained.


Visual Effects & Flashing

No rapid flashing (more than 3 Hz) that could trigger photosensitive epilepsy.
Screen transitions and scene changes smooth (no jarring cuts).
Optional reduced motion setting (disables non-essential animations).
Flashing effects used only with narrative justification (rare).


Hearing & Deafness
Subtitles & Captions

Subtitles available for all dialogue (default on, toggleable).
Captions include speaker identification (especially important for off-screen dialogue).
Captions describe important audio cues (e.g., "[Door creaks]", "[Distant voice]").
Subtitle timing precisely synced to audio.


Subtitle Customization

Font size adjustable: Small / Medium / Large.
Background opacity toggleable (improves readability for some).
Font selection options (if multiple fonts supported).
Subtitle position on screen adjustable (player preference).


Audio Descriptions (Optional)

Optional audio description track for key visual moments.
Describes important environmental details during pauses in dialogue.
Helps blind/low-vision players understand locations and interactions.


Critical Audio Cues
All gameplay-critical audio has visual equivalent:

Dialogue always captioned.
Important ambient audio (footsteps, door opening) shown with visual indicator.
No critical information conveyed by sound alone.


Motor & Dexterity
Input Remapping

All controls fully remappable (keyboard and controller).
Customizable keybinds for every action (interact, inventory, pause, etc.).
Preset control schemes for common setups (WASD, Arrow Keys, Controller).


Interaction Modes

Hold to Interact — requires holding input key (may be easier for some).
Toggle to Interact — single press to activate (may be easier for others).
Player chooses preferred mode in settings.


Sensitivity & Response

Mouse sensitivity adjustable (for keyboard + mouse).
Controller sensitivity adjustable (look speed, movement speed).
Deadzone settings for analog sticks (helps with stick drift).
No time-sensitive quick-time events that punish slower input.


Input Alternatives
Support for:

Keyboard only (no mouse required).
Controller only (no keyboard required).
Mouse + Keyboard combination.
Optional eye-tracking devices (if accessible input library supports).


Movement & Navigation

Walking speed reasonable (player never forced to run or move fast).
No fall-damage or punishing environmental hazards.
Accessible pathways around obstacles (no mandatory platforming or precise jumps).
Clear visual navigation aids (no invisible walls or confusing layouts).


Cognitive & Neurodevelopmental
Clarity & Simplicity

UI language clear and straightforward (avoid jargon).
Instructions explicit and not requiring inference.
Objectives stated clearly (no vague or unclear goals).
Tutorial comprehensive but optional (always available).


Navigation & Orientation

Consistent menu structure (predictable navigation).
Clear pathfinding (areas logically connected, easy to find destinations).
Optional breadcrumb hints (shows recently visited areas).
No complicated puzzle solutions that require external notes (information provided in-game).


Pacing & Time Pressure

No time limits on puzzles or objectives (exploration-based game naturally slow).
Players can take as long as needed to read, listen, and think.
No countdown timers or urgent pressure mechanics.


Difficulty & Challenge

Single difficulty mode (ROOTS is narrative-focused, not mechanically challenging).
Puzzles have clear logic (solvable by deduction, not trial-and-error).
No permadeath or save-scumming requirements.


Mobility & Accessibility Equipment
Voice Commands (Optional)

If implemented, voice commands for common actions (open inventory, pause, etc.).
Always optional (not required for gameplay).
Respects player privacy (no mandatory voice recording or monitoring).


Adaptive Controller Support

Game supports Xbox Adaptive Controller (if on Xbox platform).
Fully customizable input mapping supports various adaptive devices.


Comfortable Play Duration

No mandatory long play sessions without breaks.
Save system allows frequent saves (player can quit anytime).
No achievement/stat penalties for taking breaks.


General Accessibility Standards
Game Localization

Subtitles available in multiple languages (if resources allow).
UI text translatable (not hardcoded images).


Documentation & Help

In-game help/tutorial covering all systems.
Settings menu explains each option clearly.
No assumption of player prior gaming experience.


Testing & Feedback

Accessibility features tested with actual players (including disabled players).
Feedback from accessibility community incorporated.
Known limitations documented (if any features impossible to support).


Forbidden Accessibility Practices
Do not include:

Essential information conveyed by color alone.
Rapid flashing or seizure-inducing effects.
Unskippable cutscenes or tutorials.
Text too small to read comfortably.
Mandatory time pressure or quick reflexes.
No subtitle or audio description options.


Final Rule
Accessibility in ROOTS is not a compromise — it is a commitment to ensuring that every player, regardless of ability, can fully experience and enjoy the story, world, and investigation at the heart of the game.
