# FONTS

## Purpose

This document defines the standards for all fonts used in ROOTS.

Typography should prioritize readability, consistency, and immersion across the entire game.

---

# Philosophy

Fonts should communicate information clearly without drawing attention to themselves.

Simple, modern typography is preferred over decorative styles.

---

# Font Categories

## Primary Font

Used for:

* Menus
* Settings
* Buttons
* HUD
* Journal
* Inventory

This font should be clean and highly readable.

---

## Secondary Font

Used for:

* Headings
* Chapter titles
* Major UI sections

It should complement the primary font while maintaining consistency.

---

## Story Font

Used only when appropriate for in-game documents.

Examples:

* Handwritten letters
* Diary pages
* Old notes

It must remain readable.

---

## Monospace Font

Used for:

* Computer terminals
* Password screens
* Archive systems
* Technical documents

---

# Readability

Fonts must remain readable at:

* Small UI sizes
* Large titles
* High-resolution displays
* Different screen sizes

Avoid thin or overly decorative fonts.

---

# Language Support

Fonts should support all languages planned for ROOTS.

Minimum support:

* English
* Numbers
* Common punctuation
* Extended Latin characters

Additional language support can be added later.

---

# Text Styling

Use styling sparingly.

Allowed:

* Bold
* Italic
* Different sizes

Avoid:

* Decorative shadows
* Outlines
* Excessive gradients
* Flashing text

---

# File Formats

Preferred formats:

* `.ttf`
* `.otf`

Keep the original font files in the project.

---

# Optimization

* Include only required font weights.
* Remove unused font variants.
* Cache frequently used fonts.
* Avoid loading unnecessary fonts.

---

# Naming Convention

Examples:

```text id="m73x9v"
font_primary.ttf
font_secondary.otf
font_story_handwritten.ttf
font_terminal.ttf
```

Use lowercase letters and underscores only.

---

# Design Rules

* High readability.
* Consistent typography.
* Minimal font families.
* Accessible sizes.
* Professional appearance.

---

# Final Rule

Typography in ROOTS should quietly support the experience. Players should effortlessly read menus, documents, and dialogue while remaining fully immersed in the world.
