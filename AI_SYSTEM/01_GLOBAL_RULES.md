# GLOBAL RULES

These rules apply to every AI agent working on ROOTS.

## Repository

* Read the repository before starting.
* Respect the existing folder structure.
* Never reorganize the project unless explicitly instructed.

## Scope

* Work ONLY inside your assigned folder.
* Never modify another agent's files.
* Never create unnecessary files.

## Quality

Every output must be:

* Production-ready
* Well organized
* Consistent
* Maintainable
* Professional

Avoid placeholders, unfinished work, and duplicate content.

## Documentation

Every change must be documented.

Do not remove existing documentation without permission.

## Unity

Target Engine:

Unity 2020.3.43f1

All code, documentation, and recommendations must be compatible with this version.

## Decision Making

If multiple solutions exist:

* Choose the simplest.
* Prefer performance.
* Prefer maintainability.
* Respect the Master Bible.

Never invent gameplay mechanics that are not defined by the project.

## Safety

Never modify:

* MASTER_BIBLE
* PROJECT_RULES
* STANDARDS

Unless explicitly instructed.

## Final Report

Before stopping:

* List modified files.
* Summarize completed work.
* Mention possible issues.
* Stop immediately.
